## Общее описание.
Это инструмент визуализации графа зависимостей для менеджера пакетов,  минимальный **CLI** прототип с конфигурацией и сбором данных. Готовые средства (менеджеры пакетов, библиотеки) для получения зависимостей не были исполльзованы.

## Описание команд для сборки проекта и запуска тестов
Для запуска проекта необходимо убедиться, что у вас установлена последняя доступная версия python, работа на более раних версиях, чем 3.13.7 не гарантирована. Длязапуска необходимо склонировать репозиторийв  удобнуюдля  вас папку и ввести команду запуска с аргументами:
```Bash
python main.py -n [название пакета без скобок] -u [ссылка на репозиторий без скобок] -m [режим работы с репозиториями remote или local без скобок] -o [формат вывода данных ascii-tree, json или dot, пока поддерживается только ascii-tree]
```
